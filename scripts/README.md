# Running manual Textworld game from

``` bash
  # Random game at difficulty 1
  python run_manual_textworld_cooking.py

  # Specific game by id
  python run_manual_textworld_cooking.py --game-id 5

  # Change difficulty (1-10)
  python run_manual_textworld_cooking.py --difficulty 3

  # Use train split
  python run_manual_textworld_cooking.py --split train --game-id 0
```
